## üöÄ How to Use

### Creating a Mind Map
1. Click "New Mind Map" from dashboard
2. Enter a title
3. The root node is created automatically
4. Click "Add Node" to create new nodes
5. Drag nodes to position them
6. Click between nodes to create connections

### Styling Nodes
1. Click on a node to select it
2. Use the color palette in toolbar to change color
3. Use the shape dropdown to change shape
4. Click on node text to edit

### Exporting
1. Click PNG, JPG, or SVG button in toolbar
2. The mind map will download automatically

### Saving
1. Click the "Save" button in the header
2. Your mind map is saved to MongoDB
3. Access it anytime from the dashboard
